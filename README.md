# ProjetoCrud-ControleEstoque-CadastroClientes
# WIP IMPORTS

Simulação de uma loja de roupas on-line com sistema de cadastro de produtos, cadastro de usuários/vendedores e uma loja completa com sistema de adicionar ao carrinho .
O sistema é construido inteiramente em HTML5, JAVASCRIPT e CSS3.
O sistema está sendo construido para um projeto da primeira fase do SESI SENAI, com o objetivo de avaliar nossas capacidades técnicas e criativas.
